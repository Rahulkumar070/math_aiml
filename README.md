## How to run anything in this repo

```bash
# 1. Create a virtual environment (once)
python3 -m venv .venv

# 2. Activate it (every time you open a new terminal)
source .venv/bin/activate

# 3. Install the libraries (once, after activating)
pip install -r requirements.txt

# 4. Run a file
python 00-python-basics/01_hello.py
```
